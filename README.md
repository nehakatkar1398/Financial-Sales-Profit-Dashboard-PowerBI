# 📊 Financial Sales & Profit Dashboard (Power BI)

# 🔹 Project Overview
This project showcases an interactive Power BI dashboard designed to analyze business performance using sales and profit metrics. It includes a navigation-based structure for better user experience.


# 🔹 Dashboard Structure - 

# 🧭 Summary Page
* Central navigation page
* Buttons to switch between dashboards
* 

# 📈 Sales Dashboard
* KPI Cards: Total Sales, COGS, Profit, Profit Margin
* Sales Trend Analysis
* Country-wise Performance
* Product-wise Insights
* Segment-wise Insights

# 📊 Profit Dashboard
* KPI Cards: Total Sales, COGS, Profit, Profit Margin
* Profit trends
* Region-wise comparison
* Segment analysis
* Product-wise Insights
  
# 🔹 Tools Used
* Power BI
* Data Visualization  
* DAX

# 🔹 Key Highlights
* Interactive navigation between pages
* Clean and modern UI
* Business-focused insights
* Navigation buttons for seamless user experience
* External website link integration
